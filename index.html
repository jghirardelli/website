Scientific Computing Projects

These projects demonstrate my experience using programming to solve and visualize physics problems.

C++ Damped Harmonic Oscillator

A numerical simulation of a damped driven oscillator using C++. The project models position, velocity, and energy over time and compares numerical behavior with expected physical trends.

Concepts: ODEs, numerical integration, energy analysis

Tools: C++, Python/Matplotlib for visualization

GitHub Repository

1D Heat Equation Solver

A computational model for solving the one-dimensional heat equation using finite difference methods and tridiagonal matrix techniques.

Concepts: PDEs, boundary conditions, numerical stability

Tools: C++, Python visualization

GitHub Repository

2D Electrostatic Potential Solver

A relaxation-method solver for Laplace’s equation in two dimensions, used to model electric potential and electric fields for conducting geometries.

Concepts: Laplace’s equation, relaxation methods, electric fields

Tools: C++, Python visualization
